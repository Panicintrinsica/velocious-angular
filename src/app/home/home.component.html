Welcome to Velocious

This boilerplate application is intended to jumpstart projects that require user authentication up and running faster. This app, along with it's companion API, provide a basic framework which you can extend as needed.

The project is intentionally bare bones, and uses minimal styling to make it easier to use whatever component library you wish. It is assumed you will be rebuilding every page for you specific needs, and as such the provided templates exist only as demonstrations of functionality.

Current Features Include

  • Account Registration
  • E-Mail Activation
  • Password Resets
  • Auth & Refresh Tokens
